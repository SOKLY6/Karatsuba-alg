# Полиномы и алгоритм Карацубы

Проект для работы с полиномами: вычисление значений, умножение классическим способом и методом Карацубы.

## Активация виртуального окружения

Прописать команду после `make build`

`. .venv/bin/activate`

## Команды Makefile

- `make build` - создание виртуального окружения и установка зависимостей
- `make test` - запуск тестов производительности, результат сохраняется в `report.txt`
- `make start` - запуск интерактивного CLI для работы с программой

## Использование CLI

После запуска `make start` выберите режим работы:
1. Прямое вычисление значения полинома
2. Вычисление по схеме Горнера
3. Классическое умножение полиномов
4. Умножение полиномов методом Карацубы
5. Умножение чисел (через полиномы)

Коэффициенты полиномов вводятся через пробел, начиная со старшей степени.
